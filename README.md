# Аналитика и аналитическая инфраструктура в сфере информационной безопасности

Практический курс для студентов старшего курса УГТУ-УПИ, факультет информационной безопасности.

## О курсе

В ходе курса вы постепенно развернёте на локальной машине аналитическую инфраструктуру и научитесь применять её для анализа данных в сфере информационной безопасности.

__Продолжительность:__ 32 академических часа (16 практических занятий)

__Ролевая модель:__ специалист по информационной безопасности, которому необходимо настроить сбор, сохранение и аналитику данных.

## Формат заданий

Курс использует комбинированный подход:

- Начинаем с шаблонов и готовых примеров
- Постепенно переходим к самостоятельному написанию кода с нуля
- Каждый блок содержит теоретический материал, практические примеры и задания

## Требования

- Docker и Docker Compose
- UV (для управления зависимостями)
- Python 3.14+
- 8 ГБ оперативной памяти

## Начало работы

1. Склонируйте репозиторий
2. Установите зависимости: `uv sync`
3. Перейдите к первому блоку

## Структура репозитория

```
├── lessonNN/          # Материалы блока (теория, примеры, данные)
├── solutions/         # Эталонные решения (приватный репозиторий)
└── tests/             # Тесты для проверки заданий
```

## Проверка заданий

Курс использует комбинированный подход к проверке заданий:

Для запуска тестов используйте pytest:

```bash

# и т.д.
```
